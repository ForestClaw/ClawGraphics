### Welcome to Claw Graphics! 

---

This is a repository of Matlab routines, developed over the years, that support visualization of Clawpack, AMRClaw and ForestClaw output.   Output is dessigned specifically to support ASCII output from these packages.   For more information on Clawpack and AMRClaw see [http://www.clawpack.org](http://www.clawpack.org).  

For information on how to use ClawGraphics in particular, see   and [https://www.clawpack.org/matlab_plotting.html](https://www.clawpack.org/matlab_plotting.html).   F
