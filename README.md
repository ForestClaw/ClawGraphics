### Welcome to ClawGraphics! 

---

This is a repository of Matlab routines, developed over the years, that support visualization of Clawpack, AMRClaw and ForestClaw output.   Output is dessigned specifically to support ASCII output from these packages.   

For information on how to use ClawGraphics see   and [https://www.clawpack.org/matlab_plotting.html](https://www.clawpack.org/matlab_plotting.html). 

For more information on Clawpack and AMRClaw see [http://www.clawpack.org](http://www.clawpack.org).  

For information on ForestClaw, see [http://www.github.com/ForestClaw/forestclaw].
